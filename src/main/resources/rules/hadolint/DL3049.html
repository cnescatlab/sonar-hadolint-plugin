Rationale :

This rule is only activated, when specifying a label-schema to Hadolint. This example presumes the label-schema is: 

label-schema:
    version: text
or specified via command line: 
$ hadolint --require-label version
When a label-schema is specified, all labels within that schema must be present.

Problematic code :

FROM busybox

Correct code :

FROM busybox
LABEL version="1.0.1"
