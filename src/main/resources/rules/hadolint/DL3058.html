Rationale :

This rule is activated by specifying a label-schema including a label of type email. This can be specified via config file : 

label-schema:
  maintainer: email
or via command line: 
$ hadolint --require-label maintainer:email Dockerfile

Problematic code :

LABEL maintainer="foo-bar"

Correct code :

LABEL maintainer="foo@bar.com"

Note :

This rule does not expand variables. It requires the value of a label to be a literal email compliant with RFC 5322.
