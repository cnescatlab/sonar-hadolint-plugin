Rationale :

This rule is activated by specifying a label-schema including a label of type rfc3339. This can be specified via config file : 

label-schema:
  created: rfc3339
or via command line: 
$ hadolint --require-label created:rfc3339 Dockerfile

Problematic code :

LABEL created="today"

Correct code :

LABEL created="2021-03-23 13:04:29.300508335+01:00"

Note :

This rule does not expand variables. It requires the value of a label to be a literal time stamp compliant with RFC 3339.
